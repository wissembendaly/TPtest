Welcome to WatchNear

WatchNear est un site web qui a pour but de donner plusieurs informations sur les salles de cinémas et les films pour aider les utilisateurs à choisir la salle qui leur semble la plus adéquate. ♦ Faire la personnalisation du profil de l'utilisateur (changement de photo de profil, d'email…) ♦ Effectuer une recherche d'un film pour ensuite acquérir les informations nécessaires pour y procéder. ♦ Consulter les informations nécessaires pour presque toutes les films qui sont déjà diffusés ou en cours de diffusion. ♦ Voir le Planning de chaque salle de cinéma pour chaque semaine ou en choisissant les dates. ♦ Contacter le support pour s'interroger sur le fonctionnement du site ou donner un avis.
